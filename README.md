# IP-TOOLS
 This Tool is a collection of IP Tools that can be used to quickly get information about IP Addresses, DNS records and store the information in a pdf with user friendly format.

## Features:
### WHOIS
### Traceroute
### DNS	LOOKUP
### REVERSE	DNS
### DNS	HOST	RECORDS
### GEOIP	LOOKUP
### SUBDOMAINS
